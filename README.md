# co-scientist-cloud

Firebase-based fork of [co-scientist](https://github.com/k821209/co-scientist).

Local MCP for Claude Code + Firestore/Storage + static SPA on Firebase Hosting.
No Cloud Run, no server-side LLM. Heavy compute stays on the user's machine or
their registered HPC.

See `apps/local-mcp/` for the MCP server and `tests/` for smoke tests.
